## 2. 通用接口结构

每个报告字段先按 `指标名称 + 指标路径 + 指标数据.日期类型` 找到唯一记录，再从该记录的 `指标数据` 中取值。

| 标准字段 | JSON 字段 | 说明 |
| --- | --- | --- |
| `metric_name` | `指标名称` | 指标名 |
| `metric_path` | `指标路径` | 指标完整业务路径，严格匹配 |
| `period_type` | `指标数据.日期类型` | `月` / `季` / `年` |
| `actual_value` | `指标数据.实际值` | 本期实际值 |
| `target_value` | `指标数据.目标值` | 目标值 |
| `prior_value` | `指标数据.同期数` | 去年同期值 |
| `gap_value` | `指标数据.扣分值` | 本章过程指标中作为差距使用 |
| `completion_rate` | `指标数据.达成率` | 接口原始百分数，不再乘 100 |
| `unit` | `指标数据.单位` | 万、个、家、吨、千平米、% 等 |
| `division_rank` | `指标数据.分部排名` | 分部排名，本 PDF 未展示 |
| `province_rank` | `指标数据.省区排名` | 省区排名，本 PDF 未展示 |
| `department_rank` | `指标数据.部门排名` | 部门排名，本 PDF 未展示 |

## 7. 对接注意事项

1. 禁止按数组下标取值；接口数组顺序变化后会串字段。
2. 禁止只按 `指标名称` 匹配；同名指标可能出现在不同路径或日期口径。
3. `达成率` 已是百分数，例如 `50.000` 应展示为 `50.000%`，不要再次乘 100。
4. 产品和行业是动态数组，不能把具体产品名或行业名写死成固定列。
5. `有效项目落地` 是接口名称，PDF 展示名为“有效落地项目”。
6. 本 PDF 中“有效落地项目”月口径单位为“家”，季/年口径单位为“个”，属于上游单位不一致，应由数据源确认，映射层不要静默改写。
