## 1. 映射约定

1. 记录只能用 `指标名称 + 指标路径 + 指标数据.日期类型` 精确命中。
2. 不使用数组位置、数值大小、相似名称、Top N 排序或缺省日期推断。
3. 直接取值保留 JSON 原始精度，只追加同一记录的 `指标数据.单位`。
4. 精确条件无命中时，报告显示红色“待补充”，不使用其他字段自行计算。
5. 精确条件命中多条且数值不一致时，报告显示红色“待补充”，核对清单记录冲突候选值。

## 4. 3.3 产品、客户、项目及行业

| 报告内容 | 匹配条件 | 取值字段 | 处理/公式 |
| --- | --- | --- | --- |
| 正向产品 | `各产品销量` 年口径 | `实际值` / `同期数` / `单位` | 先筛选收入同比正增长，再按本期收入降序取前三；增长率=`(实际值-同期数)/同期数×100%` |
| 正向客户 | `指标名称="产销客户数"` 且年口径 | `实际值` / `同期数` | 增长率同上；增加个数=`实际值-同期数` |
| 正向项目 | `指标名称="产销项目数"` 且年口径 | `实际值` / `同期数` | 增长率同上；增加个数=`实际值-同期数` |
| 正向行业 | 同名 `各行业销量` 与 `各行业销量占比` 年口径 | 两条记录的 `实际值` / `同期数` | 筛选收入和占比均增长的行业，按本期占比降序取前三；占比增长=`本期占比-同期占比` |
| 风险产品 | 同名 `各产品销量` 与 `各产品销售量` 年口径 | 两条记录的 `实际值` / `同期数` | 筛选收入与销售量均下降的产品，按本期收入降序取前三 |
| 风险客户 | `指标名称="客均销量"` 且年口径 | `实际值` / `同期数` | 计算同比下降率；客户明细未提供，该位置红色“待补充” |
| 风险项目 | `指标名称="单项目销量"` 且年口径 | `实际值` / `同期数` | 计算同比下降率 |
| 风险行业 | 同名 `各行业销量` 与 `各行业销量占比` 年口径 | 两条记录的 `实际值` / `同期数` | 筛选收入和占比均下降的行业，按本期占比降序取前三 |

## 6. 计算清单

报告 3.2 对打样项目数展示两类代码计算数字：差距=`实际值-同期数`；增长率=`(实际值-同期数)/同期数×100%`。3.3 产品、客户、项目和行业同比增长/下降率使用同一公式；占比增长/下降使用 `|本期占比-同期占比|`。所有计算百分比四舍五入保留2位小数。
