## 1. 客户 Word 模板结构

模板固定文字仅包括：

模板没有要求加入费用合计、费用收入比或自动归因文字，本章不派生这些数字。

## 2. 唯一匹配契约

每个报告数字必须同时精确匹配：

```text
指标名称 == 契约中的完整名称
指标路径 == "二、利润概况-" + 完整指标名称
指标数据.日期类型 == "月" | "季" | "年"
```

命中后仅取：

```text
数值：指标数据.实际值
单位：指标数据.单位
```

不使用数组顺序、数值大小、相似名称或单位语义猜测。不改写 `实际值` 字符串的小数精度。

## 4. 缺失、重复和冲突

| 情形 | 报告处理 |
| --- | --- |
| 精确条件无记录 | 红色“待补充” |
| `实际值` 或 `单位` 缺失 | 红色“待补充” |
| 同一精确条件多条记录且值/单位不同 | 红色“待补充”，保留全部候选证据 |
| 同一精确条件多条记录且值/单位完全一致 | 可展示共同值，状态记为“重复（值一致）” |
| 接口单位与 Word 模板单位冲突 | 除已确认的毛利率口径外，不做单位换算，红色“待补充” |
